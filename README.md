# AtMail

:e-mail: A web application to see, reply, has its mails received in a database

# Dependencies
```shell
$ npm install
```

# How to hack
```
$ npm run start
```

# How to build
```
$ npm run build
```

# Configuration
```json
{
    "database": {
        "host": "",
        "database": "",
        "user": "",
        "password": "",
    }
}
```
